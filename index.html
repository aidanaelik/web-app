Оқушыларға арналған веб-приложение

Сабақ тақырыбы: HTML негіздері

HTML – веб-беттер жасауға арналған тіл. Ол "HyperText Markup Language" дегенді білдіреді.

Толық ақпарат: HTML Википедия

HTML құрылымы келесідей:

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Мысал беті</title>
                </head>
                <body>
                    <h1>Сәлем, әлем!</h1>
                </body>
                </html>
