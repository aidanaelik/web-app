Оқушыларға арналған веб-приложение

Сабақ тақырыбы: HTML негіздері

HTML – веб-беттер жасауға арналған тіл. Ол "HyperText Markup Language" дегенді білдіреді.

Толық ақпарат: HTML Википедия

HTML құрылымы келесідей:

                <!DOCTYPE html>
                <html>
                <head>
                    <title>Мысал беті</title>
                </head>
                <body>
                    <h1>Сәлем, әлем!</h1>
                </body>
                </html>

Тест сұрағы

HTML-де ең негізгі тег қандай?

CSS негіздері

CSS – Cascading Style Sheets, веб-беттің дизайнын жасау үшін қолданылады.

Толық ақпарат: CSS Википедия

Мысалы, CSS коды:

                body {
                    background-color: lightgray;
                    font-family: Arial, sans-serif;
                }
                h1 {
                    color: blue;
                }

CSS бойынша тест

CSS-те түсті өзгерту үшін қандай қасиет қолданылады?
